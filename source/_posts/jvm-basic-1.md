## JVM 是什么

JVM 全称是 Java Virtual Machine，中文名为 Java 虚拟机，是一个运行在计算机上的程序，他的职责是运行 Java 字节码文件。

## JVM 的功能

1. 解释和运行

   - 对字节码文件中的指令，实时的解释成机器码，让计算机执行
2. 内存管理

   - 自动为对象、方法等奉陪内存
   - 自动的垃圾回收机制，回收不再使用的内存
3. 即时编译

   - 对热点代码进行优化，提升执行效率

### 即时编译

{% asset_img 20241021213823.png %}

JVM 提供了 即时编译（JIT）进行性能优化。最终能达到接近 C/C++ 语言的运行性能。在特定场景下甚至实现超越。
