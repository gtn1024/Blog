# 基础语法

## 语言标记

PHP中，PHP代码通常以`<?php`开始，以`?>`结束。如：

```php
<?php
  // ...
  ?>
```

如果直到文件最后均为php代码，则`?>`可以省略。如：

```php
<?php
  // ...
```

## 语句分隔

在PHP中，每条语句的结束需要以`;`结尾。如：

```php
<?php
  echo "Hello World!";
?>
```

特别地，如果语句仅有一行，则分号可以省略。如：

```php
<?php echo "Hello World!" ?>
```

## 注释

PHP支持多种注释风格。

单行注释：可使用`//`、`#`表示

多行注释：使用`/* */`表示

如：

```php
<?php
  // 这是单行注释
  # 这也是单行注释
  
  /*
  这是
  多行
  注释
  */
  
```

文档注释：使用`/** */`表示
